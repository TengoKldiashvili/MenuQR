# Digital Menu Builder

A modern SaaS platform for creating beautiful digital restaurant menus with QR code sharing.

## Features

- User authentication and registration
- Create and manage multiple menus
- Add categories and menu items
- Upload images for menu items
- Generate QR codes for public menu sharing
- Multiple theme options (light, dark, minimal, elegant)
- Responsive design

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Prisma ORM
- PostgreSQL (Neon DB)
- NextAuth.js
- UploadThing for image uploads
- TailwindCSS
